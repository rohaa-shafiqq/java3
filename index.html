Legal And Illegal Variables

Rules for naming JS variables

Variable names can only contain letters, numbers, and underscores (_). For example: Smy_1st

Variables must begin with a letter, underscore (_), or dollar sign ($). For example: Sname, _name, or name

Variable names are case sensitive. For example: name and Name are different variables.

Variable names should not be JavaScript reserved keywords like var, let, function, etc.
